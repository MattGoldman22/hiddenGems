About Us:

What is Hidden Gems?

Hidden Gems is an online database for all the "hidden" areas around Arizona.

Why did we start Hidden Gems?

We started Hidden Gems as a class project because we wanted to provide a place where incoming and current students can find unique spots to see around each ASU campus.

How did we start Hidden Gems?

We started by gathering a list of all of our favorite places around Arizona that we found special that not many people knew about and collaborated to put into an easy to access website.
